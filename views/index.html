URL Shortener Microservice

User Stories

  1. I can pass URL as a parameter and I will receive a shortened URL in the JSON response.
  2. If the URL does not follow the valid format
    http://www.exaple.com/ or https://www.exaple.com/, the JSON response contains an error instead.
  3. I can visit the sortened URL, it will redirect me to my original link.

Examples of Usage

  • Type https://alert-punishment.glitch.me/api/shorten/https://github.com
  • or https://alert-punishment.glitch.me/api/shorten/http://www.google.com

Example output

Success: { "long_url": "https://github.com", "short_url": "http://something-fancy.glitch.me/i4" }
Error: { "error": "Wrong url format. Please enter a valid url", "exapmle": "http://www.exapmle.com" }

Example of usage

Type https://alert-punishment.glitch.me/hY
and you will be redirected to https://www.github.com/
